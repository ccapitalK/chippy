# chippy
A Chip8 emulator I wrote while learning rust

## Usage
    chippy [rom file]

## controls
    Key 0 - 4
    Key 1 - 5
    Key 2 - 6
    Key 3 - 7
    Key 4 - R
    Key 5 - T
    Key 6 - Y
    Key 7 - U
    Key 8 - F
    Key 9 - G
    Key A - H
    Key B - J
    Key C - V
    Key D - B
    Key E - N
    Key F - M

## Sound
Sound is currently not implemented
